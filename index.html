RECESS

Final project at Dev Bootcamp- a web app built in Ruby that is a tool to help groups plan travel itineraries. Owned the Google Maps and Places API build and integration. The app includes the capability to search for locations and pull in information via the Google Places API. The Maps API was used to display a customized Google Map that showed the relevant places information. This app was entirely planned for, developed and launched over the course of five days.

HERSHEY'S SELF SERVICE FRAMEWORK

Led and managed product vision for the concept and build of Hershey's AEM Platform. The scope of this work included the UX conception and build of a module library, UI and Platform code integration, Platform training for Hershey brands and the creation of framework process. This nine month-long project required management of a full cross-discipline team (UX, Visual Design, UI, Platform) and owning vision from the smallest user interaction on the platform to the creation of the operational process.

MCCANN GALLERY APP
